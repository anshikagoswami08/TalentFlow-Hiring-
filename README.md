# 🚀 TalentFlow - Modern Hiring Platform

> A modern, full-featured hiring platform built with React, TypeScript, and Vite. Manage candidates, create assessments, and streamline hiring with an intuitive Kanban interface.

---

## 🌟 Live Demo

🔗 https://talentflow-hiring.vercel.app

---

## ✨ Features

### 🎯 Core Functionality

- Candidate Management System
- Kanban Board for Hiring Pipeline
- Job Posting & Tracking
- Assessment Builder
- Real-time Search & Filtering
- Notes with Mentions

### 🎨 UI/UX

- Responsive Design
- Dark/Light Mode
- Smooth Animations
- Clean Dashboard Layout

### 🔧 Tech Features

- TypeScript Support
- React Query for Data Handling
- Local State Persistence
- API Mocking (MSW)

---

## 🛠️ Tech Stack

- React (18)
- TypeScript
- Vite
- Tailwind CSS
- React Query
- Framer Motion

## 🚀 Installation

```bash
# Clone repo
git clone https://github.com/anshikagoswami08/TalentFlow-Hiring-.git

# Go inside folder
cd TalentFlow-Hiring-

# Install dependencies
npm install --legacy-peer-deps

# Start project
npm run dev
```

Open: http://localhost:8081
